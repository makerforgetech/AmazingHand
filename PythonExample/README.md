## Rock, Paper, Scissors game

Install virtual environment
```bash
python3 -m venv myenv
source myenv/bin/activate
```

Install dependencies
```bash
python3 -m pip install -r PythonExample/requirements.txt
```

Run the game
```bash
python3 RockPaperScissors.py
```
